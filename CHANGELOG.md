# Changelog

## [1.5.7] - 2026-04-16
### Added
- Добавлен пользовательский скрипт `Russian-Uncensored-Words-Highlight-1.5.7.user.js`

### Changed
- Обновление версии до 1.5.7
- Обновлена документация (README.md)

## [1.5.6] - 2026-04-03
### Added
- Добавлены недостающие паттерны: `HUYN`, `PITАR`

### Changed
- Упорядочивание паттернов по алфавиту для улучшения читаемости кода
- Сортировка символов внутри символьных классов регулярных выражений по парам "строчная+прописная" для удобства восприятия (например, [аАмМюЮ])
- Исправлена обработка escape-последовательностей (например, `\u0020`) при сортировке
- Паттерн `BLYA` разделён на `BLYAD` и `BLYAT` для более точного поиска

## [1.5.5] - 2026-03-23
### Added
- Добавлены паттерны: `HUY`, `PLYA`, `IBI`
- Добавлена обработка дополнительных тегов: `blockquote`, `code`

### Changed
- Улучшена производительность регулярных выражений
- Уменьшена нагрузка на процессор

### Fixed
- Исправлены ложные срабатывания для слов "питер", "хлеб"
- Исправлена обработка текста внутри iframe

## [1.5.4] - 2026-03-22
### Added
- Добавлены паттерны: `PITER`, `PISD`
- Паттерн `EBA` теперь находит "ебобо"
- Добавлена обработка тегов `p, a, li, td, th, em, strong, b, i`

### Changed
- Интервал проверки увеличен до 5 секунд
- Убрано зависание страницы

## [1.5.3] - 2026-03-21
### Added
- Добавлены паттерны: `MAND`, `PIZH`
- Добавлена обработка тегов `span`, `div`

### Fixed
- Исправлены ложные срабатывания для слов "хлеб", "пирог"

## [1.5.2] - 2026-03-20
### Added
- Добавлены паттерны: `HUI`, `ZALUPA`
- Добавлена поддержка визуальных замен символов (3, 6, 0, @, ×, y, p, $, ¥, α, ë)

### Changed
- Улучшена точность контекстных проверок

## [1.5.1] - 2026-03-19
### Added
- Первая публичная версия
- Базовые паттерны: `DOLBO`, `PIDOR`, `PIZD`, `EBA`, `EBLO`, `IPA`, `BLYAD`, `BLYAT`
- Подсветка жёлтым маркером
