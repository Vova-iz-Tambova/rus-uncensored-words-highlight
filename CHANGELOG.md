# Changelog

## [1.6.7] - 2026-05-26
### Fixed
- Исправлена ломка flex/grid-контейнеров: убран `display: contents` у враппера, теперь обычный `<span>` не создаёт несколько flex-элементов
- Исправлены ложные срабатывания в CSS/JS-коде: добавлен пропуск тегов `STYLE`, `SCRIPT`, `SVG` в TreeWalker

### Changed
- Удалены все `console.log` из production-кода
- Задержка MutationObserver сокращена до 10 мс
- Периодическая проверка `setInterval` оставлена на 300 мс как fallback для чатов

## [1.6.6] - 2026-04-28
### Added
- Добавлен символ `×` (умножение) для "х" в паттерны `HUI`, `HUY`, `HUYN`
- Добавлен символ `Ñ` (латинская N с тильдой) в паттерн `HUYN`
- Добавлена дополнительная защита от ложных срабатываний в паттернах `BLYAD`, `EBA`, `HUI`, `IPA`

### Changed
- Ускорена работа скрипта: начальная задержка сокращена до **100 мс**, интервал проверки до **300 мс**
- Улучшены контекстные ограничения в паттерне `BLYAD` (добавлены `кО`, `щО`, `оО`)
- Оптимизирован паттерн `EBA` (добавлены `ыЫ`, уточнена группа символов)
- Уточнён паттерн `HUI` (добавлены `ыЫ`, `яЯ` в запрещённые комбинации)
- Улучшен паттерн `IPA` (расширен lookbehind, добавлены `иИ`, `йЙ`, `хХ`, уточнены запреты)
- Уточнён паттерн `IBI` (заменён `иИуУ` на `еЕиИуУ`, добавлен запрет `сС`)
- Уточнён паттерн `PIDOR` (добавлен `\s` в группу, добавлен символ `4` в lookbehind)

### Fixed
- Исправлена обработка слова "ебашить" и подобных
- Исправлены ложные срабатывания на словах с "пест" и "ипа"

## [1.6.5] - 2026-04-25
### Added
- Добавлен паттерн `EB` для коротких форм ("еб", "ебё")
- Добавлена поддержка символов: `δ` (дельта) для "б", `π` (пи) для "п", `ӆ` (л с хвостом)
- Добавлено игнорирование ударений и диакритических знаков (диапазон `\u0300-\u036f`)
- Добавлена поддержка начала и конца строки через искусственные пробелы
- Добавлена функция `mapIndex` для корректного преобразования индексов при наличии ударений

### Changed
- Полностью переработана логика конфликта с другими расширениями (Highlight)
- Упрощена проверка подсветок: разделены функции `isOurHighlight` и `isForeignHighlight`
- Оптимизирован обход DOM через TreeWalker для всех текстовых узлов
- Удалён атрибут `data-highlighted` как избыточный (достаточно проверки `hl-yellow`)
- Сокращена начальная задержка до 500 мс
- Уменьшен интервал периодической проверки до 1 секунды

### Fixed
- Исправлено смещение маркера на 1 символ влево при наличии ударений
- Исправлено зацикливание и зависание страницы при конфликте с Highlight
- Исправлена ситуация, когда скрипт пропускал весь текст после первого же слова с чужой подсветкой
- Исправлена обработка слов внутри уже подсвеченных элементов
