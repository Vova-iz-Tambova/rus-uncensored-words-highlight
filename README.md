# Подсветка нецензурных слов 🇷🇺

Браузерное расширение для визуальной подсветки матерных слов без цензуры.

## 🌟 Особенности

### 🔍 Выделяет слова без цензуры
Расширение выделяет слова жёлтым маркером, но не скрывает и не заменяет их. Вы видите текст целиком, просто с визуальной пометкой.

### 🧠 Умные паттерны поиска (регулярные выражения)
В отличие от обычных расширений с готовыми списками слов, наше расширение использует регулярные выражения, что позволяет:
- Ловить любые вариации слов, включая неизвестные ранее
- Распознавать визуальные замены символов (`3алупа`, `еб@`, `×уй`)
- Фильтровать ложные срабатывания через контекст
- Не требовать постоянного обновления списка

## 🎯 Актуальные паттерны (версия 1.5.4)

| Паттерн | Примеры |
|---------|---------|
| `DOLBO` | долбоёб, долбаёб |
| `ZALUPA` | залупа, залупу |
| `PIDOR` | пидор, пидора |
| `PITER` | питер (исключение ложных срабатываний) |
| `PIZD` | пизда, пизде |
| `PISD` | писда, писдой |
| `PIZH` | пиж, пижу |
| `MAND` | манда, манде |
| `HUI` | хуй, хуя, хуё |
| `HUY` | хуй (доп. контекст) |
| `BLYA` | бля, блять |
| `PLYA` | пля, плям |
| `EBA` | ебать, ебобо, еба |
| `IBI` | иби, ибит |
| `EBLO` | ебло, еблу |
| `IPA` | ипа, ипу |

Каждый паттерн представляет собой регулярное выражение, которое учитывает возможные замены символов (например, `3` вместо `з`, `@` вместо `а`) и контекстные ограничения для минимизации ложных срабатываний. Подробности можно посмотреть в файле [patterns.js](patterns.js).

## 🎨 Распознаёт визуальные замены символов

| Символ | Заменяет |
|--------|----------|
| `3` | з |
| `6` | б |
| `0` | о |
| `@` | а |
| `×` | х |
| `y` | у |
| `p` | р |
| `H` | н |

## ✅ Что НЕ подсвечивается
- Обычные слова: `тебя`, `хлеб`, `ребёнок`, `себя`, `небо`, `команда`, `Чебурашка`
- Технические идентификаторы (хеши, ссылки)
- Уже подсвеченные другими расширениями слова (решение конфликтов)

## 📦 Установка

### Как расширение для браузера

1. Скачайте репозиторий (кнопка **Code** → **Download ZIP**)
2. Распакуйте архив в любую папку
3. Откройте браузер и перейдите в настройки расширений:
   - **Chrome/Edge/Yandex/Opera:** Меню (⋮) → Расширения → Управление расширениями → Включите **Режим разработчика** → **Загрузить распакованное расширение**
   - **Firefox:** Меню (☰) → Дополнения и темы → Расширения → Шестерёнка ⚙️ → Отладка дополнений → Загрузить временное дополнение
4. Выберите папку с расширением
5. Готово! Расширение работает на всех сайтах.
