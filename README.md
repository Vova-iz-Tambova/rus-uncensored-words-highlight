# Подсветка нецензурных слов 🇷🇺

Браузерное расширение для визуальной подсветки матерных слов без цензуры.

## 🌟 Особенности

### 🔍 Выделяет слова без цензуры
Расширение выделяет слова жёлтым маркером, но не скрывает и не заменяет их. Вы видите текст целиком, просто с визуальной пометкой.

### 🧠 Умные паттерны поиска (регулярные выражения)
В отличие от обычных расширений с готовыми списками слов, наше расширение использует регулярные выражения, что позволяет:
- Ловить любые вариации слов, включая неизвестные ранее
- Распознавать визуальные замены символов (`3алупа`, `еб@`, `×уй`)
- Фильтровать ложные срабатывания через контекст
- Не требовать постоянного обновления списка

## 🎯 Актуальные паттерны (версия 1.5.7)

| Паттерн | Примеры |
|---------|---------|
| `BLYAD` | блядь, бляди |
| `BLYAT` | блять, блят |
| `DOLBO` | долбоёб, долбаёб |
| `EBA` | ебать, ебобо, еба |
| `EBLO` | ебло, еблу |
| `HUI` | хуй, хуя, хуё |
| `HUY` | хуй (доп. контекст) |
| `HUYN` | хуйн, хуйня |
| `IBI` | иби, ибит |
| `IPA` | ипа, ипу |
| `MAND` | манда, манде |
| `PIDOR` | пидор, пидора |
| `PISD` | писда, писдой |
| `PITАR` | питар, питара |
| `PITER` | питер (исключение ложных срабатываний) |
| `PIZD` | пизда, пизде |
| `PIZH` | пиж, пижу |
| `PLYA` | пля, плям |
| `ZALUPA` | залупа, залупу |

Каждый паттерн представляет собой регулярное выражение, которое учитывает возможные замены символов (например, `3` вместо `з`, `@` вместо `а`) и контекстные ограничения для минимизации ложных срабатываний. Подробности можно посмотреть в файле [patterns.js](patterns.js).

## 🎨 Распознаёт визуальные замены символов

| Символ | Заменяет |
|--------|----------|
| `3` | з |
| `6` | б |
| `0` | о |
| `@` | а |
| `×` | х |
| `y` | у |
| `p` | р |
| `$` | с |
| `¥` | у |
| `α` | а |
| `ë` | е |

А также заглавные/строчные варианты кириллических и латинских букв.

## 🚀 Установка

1. Скачайте архив расширения или клонируйте репозиторий.
2. Откройте `chrome://extensions/` (или `edge://extensions/`).
3. Включите «Режим разработчика».
4. Нажмите «Загрузить распакованное расширение» и выберите папку с расширением.

## 📦 Структура проекта

- `manifest.json` – конфигурация расширения.
- `content.js` – основной скрипт, который выполняется на страницах.
- `patterns.js` – все регулярные выражения для поиска слов.
- `popup.html` – всплывающее окно расширения.
- `icons/` – иконки расширения.
- `Russian-Uncensored-Words-Highlight-1.5.7.user.js` – пользовательский скрипт для Tampermonkey/Greasemonkey.

## 🔧 Разработка

Паттерны регулярных выражений хранятся в `patterns.js`. Для добавления нового паттерна:
1. Создайте новую константу `PATTERN_XXXX`.
2. Добавьте её в массив внутри `regex`.
3. Протестируйте на различных текстах.
